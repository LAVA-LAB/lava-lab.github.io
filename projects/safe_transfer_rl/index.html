safe transfer RL

transfer in reinforcement learning with safety constraints

Often we need some knowledge about the safety dynamics to ensure safety during learning (Simão et al., 2021). One can precompute unsafe behavior and mask unsafe actions using a so-called shield (Jansen et al., 2020), or start from an initially safe baseline policy and gradually improve its performance while remaining safe (Achiam et al., 2017). However, this approach may require many interactions with the environment before it finds a reasonable policy.

This project investigate a setting where the agent can learn in a source task before it is deployed in the target task (Yang et al., 2022). The goal is to speed up learning on a target task while ensuring safety.

How to train a policy in a source task such that it can learn quickly and safely_ how to solve the target task?
